# di25-learnings
What we learned during the 2025 Data Institute

<p>This is a collection and recollection of what the 14 members of the <a href="https://cjddatainstitute.org/">The Data Institute</a> — a collaboration between the <a href="https://cfjd.howard.edu/">Center for Journalism & Democracy</a> and the <a href="https://idabwellssociety.org/">Ida B. Wells Society for Investigative Reporting</a> — learned July 7 to July 18 in Atlanta, Georgia.</p>
